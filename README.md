## Setup
To run the project, do this:

1) Clone project in your PC
```
$ git clone https://github.com/Grayder0152/ImageProcessing.git
```
2) Install requirements:
```
$ pip install -r requirements.txt
```
3) Run app:
```
$ python app.py
```
